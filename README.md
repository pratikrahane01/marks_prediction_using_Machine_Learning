# marks_prediction_using_Machine_Learning
Predicts student performance based on study hours, sleep, and attendance using ML.   Built with Python and scikit-learn.

# Features
- Data preprocessing and encoding  
- Multiple Linear Regression model  
- Model evaluation using R² Score and MSE  
- User input-based prediction system  

## Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

##  Result
- R² Score: 0.88  
- MSE: ~30
